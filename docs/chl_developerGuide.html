Text styles

Basic text styles

Text styles are the most basic element covered by the CHL. Styles should be defined for H1, H2, H3, H4, P, STRONG, I and BLOCKQUOTE. It is recommended that font-family, a default font-size and line-height are specified in your site style, preferably applied to the body tag.

The core styles use H6 headings for headings within rows or other nested elements.

A simple two column layout can be applied to chl2_rowCollection. The final div in each chl2_2ColumnSet must be a chl2_clear. Forms

chl2_form

CHL provides a consistent approach to form layout.

Usage:
<form class="chl2_form" action="..." [etc...]>

chl2_formItem and chl2_field

Each field and its label are grouped inside chl2_formItem. The field itself is also placed inside chl2_field. A simple implementation is shown below:

Usage:
<div class="chl2_formItem">
	<label>My label</label>
	<span class="chl2_field">
		[input, textarea, select, etc...]
	</span>
	<div class="chl2_clear"></div>
</div>

To allow targeted styling each field type also has its own class. These are:

chl2_textbox

For <input type="text" /> implement as <input class="chl2_textbox" type="text" />

chl2_password

For <input type="password" /> implement as <input class="chl2_textbox chl2_password" type="password" />

chl2_textarea

For <textarea> implement as <textarea class="chl2_textarea">

chl2_select

For <select> implement as <select class="chl2_select">

chl2_multipleSelect

For <select multiple="multiple"> implement as <select class="chl2_multipleSelect" multiple="multiple">

chl2_fileUpload

For <input type="file" /> implement as <input class="chl2_fileUpload" type="file" />

chl2_radio, chl2_checkbox and chl2_labelledField

In simple terms implement <input type="radio" /> as <input class="chl2_radio" type="radio" /> and <input type="checkbox" /> as <input class="chl2_checkbox" type="checkbox" />. However, the layout of radio buttons and checkboxes require additional markup. This is shown below:

Usage:
<div class="chl2_formItem">
	<label>My label</label>
	<span class="chl2_field">
		<span class="chl2_labelledField">
			[Radio button or checkbox]
			<label>Label for input</label>
		</span>
		[More radio buttons or checkboxes, each inside chl2_labelledField...]
	</span>
	<div class="chl2_clear"></div>
</div>

Validation and chl2_validationMessage

When validation fails chl2_error is added to chl2_formItem. A validation message is also shown (denoted by chl2_validationMessage).

Please enter a value. 
	
<div class="chl2_formItem chl2_required chl2_error">
	<label>My label <span>*</span></label>
	<span class="chl2_field">
		[input, textarea, select, etc...]
	</span>
	<div class="chl2_clear"></div>
	<span class="chl2_validationMessage">Please enter a value.</span>
</div>
