Basic CHL components

This document in conjunction with chl_theme.css describes how to add basic styling to the CHL. Developers should refer to chl_developerGuide.html.

Text styles

Text styles are the most basic element covered by the CHL. Styles should be defined for H1, H2, H3, H4 and paragraphs. It is recommended that font-family, a default font-size and line-height are specified in your site style, preferably applied to the body tag. Refer to site.css used by this example.
