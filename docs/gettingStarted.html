Getting started with CHL

This guide describes how to add CHL to a project.

How do I implement CHL?

Getting started with CHL is easy. The simplest implementation requires the addition of core.css to your project. An example implementation is shown below.

  1. <link rel="stylesheet" type="text/css" media="screen" href="my_site_style.css" />
  2. <link rel="stylesheet" type="text/css" media="screen" href="core.css" />
  3. <link rel="stylesheet" type="text/css" media="screen" href="chl2_w_subNode.css" />
  4. <link rel="stylesheet" type="text/css" media="screen" href="chl2_w_comment.css" />
  5. <link rel="stylesheet" type="text/css" media="screen" href="chl2_w_compactNews.css" />
  6. <link rel="stylesheet" type="text/css" media="screen" href="chl2_w_compactEvent.css" />
  7. <link rel="stylesheet" type="text/css" media="screen" href="chl_theme.css" />

The first style-sheet (line 1) describes the project's layout and visual style. Its name and internal structure are independent of CHL. The next style-sheet should be core.css (line 2). This file contains all of the basic styles required by CHL. If additional widgets are required their style-sheets should also be included (lines 3 to 6). The final style-sheet included is chl_theme.css (line 7). This style-sheet allows the appearance of CHL components to be customised.

Note: Do not modify core.css or the widget style-sheets. Custom CHL styles are defined in chl_theme.css (line 7).

Important: Do not include any fs_ stylesheets in your project. These are obsolete and their resets will overide CHL. If you need to update an old project, keep in mind that the fs_ stylesheets may cause unexpected results (in particular fs_text.css and fs_theme.css).

Next add a DIV with a class of content to the content area of each page. This DIV allows basic elements, such as H1s, to be styled without affecting the website's layout.

  1. <div class="content">
  2. CHL widgets and website content
  3. </div>

Copy the code from chl_testCode.html and paste the entire contents between the newly created content DIV.

You are now ready to test and style the CHL components. Refer to index.html for documentation and examples.
