Datalink CHL 2.0.15

Introduction

Datalink's Common HTML Library (CHL) allows programmers to develop websites using standard HTML that will look great on any CSS compatible site.

Designers should ensure the HTML and CSS of the websites they design follow the conventions defined in the following documents.

Developers should use the CHL when writing their modules and templates.

Developers, please note: Do not include any fs_ stylesheets in your project. These are obsolete and their resets will override CHL. If you need to update an old project, keep in mind that the fs_ stylesheets may cause unexpected results (in particular fs_text.css and fs_theme.css).

Documentation and examples

Documentation

  • Getting started shows HTML and CSS authors how to use CHL in their projects.
  • Sample components contains HTML test data that CSS authors need to style.
  • Developer guide is for programmers wishing to write templates and additional CHL widgets.

Customising CHL styles

  • An example site.css file is included in the css directory. You can copy and paste this file to the end of your own site.css. The bulk resets make it the perfect starting point for customising CHL. The file also shows how to style the tab set (you will also need to include the images/ui directory).

Examples

Testing
